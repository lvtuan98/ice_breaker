# ice_breaker

A repository for learning LangChain by building a generative ai application.

This is a web applicaiton crawling Linkedin & Twitter data about a person an customize an ice breaker with them.

Reference: 
        [github](https://github.com/emarco177/ice_breaker)
        [udemy](https://www.udemy.com/course/langchain/?referralCode=D981B8213164A3EA91AC)

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`PYTHONPATH=/{YOUR_PATH_TO_PROJECT}/ice_breaker`

`OPENAI_API_KEY`

`PROXYCURL_API_KEY`

`SERPAPI_API_KEY`

`TWITTER_API_KEY`

`TWITTER_API_KEY_SECRET`

`TWITTER_BEARER_TOKEN`

`TWITTER_ACCESS_TOKEN`

`TWITTER_ACCESS_TOKEN_SECRET`
